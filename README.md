# Project Lab

## Getting Started
* Clone the repo
```shell script
git clone https://github.com/FullStackAtBrownTeam/project-lab.git
```
### React js website 
* Navigate into the `client` directory
```shell script
cd client
```
* Install dependencies
```shell script
npm install
```
* Start the development server
```shell script
npm start 
```

### Server (Express/Node.js App)
* Navigate into the `server` directory
```shell script
cd server
```
* Install dependencies
```shell script
npm install
```
* Start your app
```shell script
npm start
```
